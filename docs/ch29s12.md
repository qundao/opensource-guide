## 数学公式

数学环境

```shell
\placeformula[1]                %给数学公式编号[引用标记]
\startformula           %数学环境起始声明
y=x^2
\stopformula            %数学环境结束声明

这是行内的数学环境 $\int_0^1 x^2 dx$    
```

### 上标、下标

使用 **`^`** 输入上标

```shell
勾股定理 $3^2 + 4^2 = 5^2$    %^2为指数
二的十次方 $2^{10} = 1024$    %如果指数不止一位，要使用{}括起来
```
